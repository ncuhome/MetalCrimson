## 资源加载包

文本结构:
``` json
{
    //是否清除之前加载的资源?
    //0: 不清除 1: 清除 2: 强制清除
    "clear":0,
    //卸载指定资源(string[]), 填入注册名
    "unload":[
        "key_1",
        "key_2",
        ...
    ],
    //加载指定资源(string[]), 填入注册名
    "load":[
        "key_1",
        "key_2",
        ...
    ],
    //强制加载指定资源(string[]), 填入注册名
    "load_force":[
        "key_1",
        "key_2",
        ...
    ],
}
```
